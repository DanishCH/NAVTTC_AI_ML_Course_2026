# NAVTTC Artificial Intelligence (Machine Learning & Deep Learning)

This repository contains all practical tasks and projects completed during the 3-month NAVTTC AI/ML course under the "Skills for All" Prime Minister's Hunarmand Pakistan Program.

## 🚀 Course Objective
The primary goal of this course is to gain employable skills in AI, with a specialization in **Natural Language Processing (NLP)** and **Microsoft Azure AI Associate** certification preparation.

## 🛠️ Tech Stack & Tools
* **Languages:** Python (Pandas, NumPy)
* **Libraries:** Scikit-learn, TensorFlow, Keras, PyTorch
* **Cloud:** Microsoft Azure API (Numerical, Image, and Text data)
* **Tools:** Anaconda, VSCode, PyCharm, Linux Shell (Debian)

## 📚 Core Competencies
* **Exploratory Data Analysis (EDA):** Hands-on practice with data cleaning and visualization.
* **Machine Learning:** Practical exposure to model design and evaluation.
* **Natural Language Processing (NLP):**
    * Text processing (Normalization, Stemming, Lemmatization)
    * POS Tagging & Named Entity Recognition (NER)
    * Word Embeddings & Deep Learning for NLP
* **Model Deployment:** Implementing and deploying NLP models.

## 📂 Repository Structure
* `Week-01/`: Linux Shell Fundamentals, Environment Setup, and Industry Surveys.
* `Week-02-04/`: Python for Data Science and EDA.
* `Machine-Learning/`: Model implementation using Scikit-learn.
* `Deep-Learning/`: Neural networks using TensorFlow and PyTorch.
* `Projects/`: Real-world applications and Azure API integrations.
